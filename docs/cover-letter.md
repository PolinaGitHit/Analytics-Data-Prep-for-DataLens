# Сопроводительное письмо

В рамках тестового задания я выполнил комплексный анализ рекламной кампании медиабайера «Delta»: проверил CPC-ставки по трём рекламным сетям, рассчитал полную ROI-воронку с 6-недельным LTV и консолидировал выгрузки из трёх разных источников в единый формат. Все расчёты выполнены на Python 3.10+ с использованием только стандартной библиотеки — без единой внешней зависимости.

Ключевые результаты: ROI кампании составил **6 230,75%** — каждый рубль рекламных затрат принёс 63 рубля выручки за 6 недель. CPC-базовый тариф 6 руб/клик подтверждён по 8 из 9 строк. CTR 3,97% значительно превышает рыночный бенчмарк. При консолидации CSV удалось нормализовать три различных формата дат, конвертировать валюты (USD → RUB по курсу 90 руб/$) и привести показы из тысяч в единицы — данные готовы для загрузки в любую BI-систему.

Работа продемонстрировала владение Python (парсинг, ETL-цепочки), аналитическое мышление (выявление аномалий, интерпретация KPI, рекомендации по оптимизации) и навыки работы с BI-инструментами (Google Sheets, Yandex DataLens). Готов обсуждать результаты и подходы на собеседовании.
