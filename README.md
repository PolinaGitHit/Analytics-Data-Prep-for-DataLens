## 📋 О проекте

Полный цикл анализа рекламной кампании: от парсинга сырых CSV-выгрузок до интерактивного дашборда с ROI-аналитикой. Проект выполнен в рамках тестового задания на позицию Media Buyer.

## 🧠 Как это работает

### Задача 1 — Анализ 3 рекламных сетей

Три скрипта с разделением ответственности:
1. **`task1_parse.py`** — парсинг CSV, расчёт CPC, проверка базовой ставки 6 руб
2. **`task1_kpis.py`** — расчёт CTR/CPM, проверка бюджетных лимитов, верификация требований
3. **`task1_recommendations.py`** — анализ использования бюджета, рекомендации на День 4

### Задача 2 — ROI через воронку LTV

**`task2_roi.py`** восстанавливает полную цепочку конверсии:

```
Затраты → Показы → Клики → Регистрации → Покупатели → Выручка (LTV 6 нед.)
```

**Результат:** ROI = 6 230.75% (каждый рубль → 63 руб выручки)

### Задача 3 — Консолидация трёх источников

**`task3_consolidate.py`** объединяет выгрузки из Яндекс.Директ, VK Реклама и SberAds:

- Разные форматы дат (ДД.ММ.ГГ, ГГ-ММ-ДД, «Март 2024, 1»)
- Две валюты (RUB и USD)
- Разные единицы (показы и показы×1000)
- Zero external dependencies — чистый Python
