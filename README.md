## Задача 1 — Анализ 3 рекламных сетей

Три скрипта с разделением ответственности:

| Файл | Назначение |
|------|-----------|
| `task1_parse.py` | Парсинг CSV, расчёт CPC, проверка базовой ставки 6 руб/клик |
| `task1_kpis.py` | Расчёт CTR и CPM, проверка бюджетных лимитов, верификация требований |
| `task1_recommendations.py` | Анализ использования бюджета, формирование рекомендаций на День 4 |

**Результат:** День 4 — повысить CPC сети РС 1 до 7 руб, лимит бюджета до 12 000 руб.

Сводка по сетям за 3 дня:

| Сеть | Показы | Клики | Затраты | CPC | CTR | CPM |
|------|-------:|------:|--------:|----:|----:|----:|
| РС 1 | 237 962 | 3 129 | 20 200 | 6.46 | 1.31% | 84.89 |
| РС 2 | 386 120 | 1 750 | 10 500 | 6.00 | 0.45% | 27.19 |
| РС 3 | 457 552 | 751 | 4 506 | 6.00 | 0.16% | 9.85 |

## Задача 2 — ROI через воронку LTV

Восстанавливает полную цепочку конверсии:

```
Затраты → Показы → Клики → Регистрации → Покупатели → Выручка (LTV 6 нед.)
                                                    ↓
                                               ROI = 6 230.75%
```

| Показатель | Значение |
|-----------|---------:|
| Общие затраты | 98 760 руб |
| Всего показов | 1 926 562 |
| Всего кликов | 76 462 |
| Регистрации | 10 440 |
| Покупатели (LTV) | 1 975 |
| **Выручка (LTV 6 нед.)** | **6 252 246 руб** |
| **ROI** | **6 230.75%** |

## Задача 3 — Консолидация трёх источников

Объединяет выгрузки из Яндекс.Директ, VK Реклама и SberAds в единый отчёт, преодолевая:

- **Разные форматы дат** — `ДД.ММ.ГГ`, `ГГ-ММ-ДД`, «Март 2024, 1»
- **Две валюты** — RUB и USD (~90 руб/USD)
- **Разные единицы измерения** — показы и показы x 1000
- **Российская локаль** — пробелы как разделители тысяч, запятые как десятичные

**Результат:** 27 строк, 20.3 млн показов, 45 тыс. кликов.

| Показатель | Суммарно |
|-----------|---------:|
| Показы | 20 305 482 |
| Клики | 45 285 |
| Затраты | 2 868 090 руб |
| Средний CTR | 0.22% |
| Средний CPM | 141.25 руб |

## Как запустить

Все скрипты используют **только стандартную библиотеку Python** — установка зависимостей не требуется.

```bash
# Задача 1 — три скрипта с разделением ответственности
python src/task1-ad-nets/task1_parse.py "src/task1-ad-nets/Тестовое маркетологу 2024, задача 1 - Лист1.csv"
python src/task1-ad-nets/task1_kpis.py "src/task1-ad-nets/Тестовое маркетологу 2024, задача 1 - Лист1.csv"
python src/task1-ad-nets/task1_recommendations.py "src/task1-ad-nets/Тестовое маркетологу 2024, задача 1 - Лист1.csv"

# Задача 2 — ROI
python src/task2-roi/task2_roi.py "src/task2-roi/Тестовое маркетологу 2024, задача 2 - Лист1.csv"

# Задача 3 — консолидация CSV
python src/task3-consolidate/task3_consolidate.py
```
