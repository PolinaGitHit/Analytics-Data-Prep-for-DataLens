## Задача 1

У вас есть [статистика](https://docs.google.com/spreadsheets/d/1Mvx8EbT2rlrOyzcYsv4ijaj-C3mfwWK1_Mv8hS83fis/edit#gid=0) по работе вашей рекламной кампании в различных рекламных сетях (РС). Известно, что:

1. Все сети работали по CPC \= 6 руб.  
2. Но в РС 1 на второй день проводили тест с поднятием ставки \- и ещё днём упёрлись в дейли-лимит по бюджету, установленный в 10т.р. для этой сети.  
3. В РС 2 стоит лимит в 3.5т.р., в РС 3 \- в 1.5т.р.  
4. В РС 2 и РС 3 все три дня кампания вставала по лимиту на бюджет ещё днём.  
5. Мы не можем оперировать креативами кампании или субисточниками ни в одной из РС.

Цель кампании \- максимум показов. Требования к кампании:

* Среднесуточный CTR по кампании, в среднем для всех сетей \> 0.5%.  
* Среднесуточный CPM по кампании, в среднем для всех сетей \< 50 руб.

Какие действия в отношении лимитов и ставок сетей вы бы предприняли на четвёртый день для выполнения цели кампании с учётом требований?

Результат выполнения этой задачи \- расчёты (включая промежуточные) в виде таблицы и рассуждения в виде сопроводительного текста.

Посчитайте итоговый ROI кампании спустя 6 недель после её окончания.

## Задача 3

У вас есть [статистика](https://docs.google.com/spreadsheets/d/11L3zlB4OPajpJoygDvLRxdXYuQpIhOOG22eY98isO3Q/edit#gid=15749816) работы ваших рекламных кампаний (РК), собранная из различных рекламных сетей (РС) в разных форматах (каждая вкладка \- выгрузка из какой-то одной РС). Сведите эту информацию в один удобный отчёт, который можно предоставить рекламодателю этих кампаний.   
Замечание \- “канонические” имена кампаний для отображения рекламодателю: “РК 1”, “РК 2”, “РК 3”.  
Просто собрать все данные на одной (новой) вкладке недостаточно, необходимо так организовать эту вкладку, чтобы отчёт легко было ежедневно расширять новыми данными с минимальными правками (а лучше вообще без них) итогового отчёта. Для этого вам пригодятся фомулы типа LOOKUP, FILTER или QUERY. При наличии опыта в каком-либо BI-инструменте (DataLens, например) идеальным решением будет создание сводного отчёта в нём.
